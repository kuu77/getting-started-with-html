What is the Web?
The Web is a collection of computers that communicate with one another remotely. A person's workstation accesses a server over the Web by sending it an HTTP request. The server finds the HTML document requested, and returns it to the user. The user's Web browser translates the HTML document into text, images, and other objects, to make it easy for the user to interpret.

What is HTML?
HTML, or Hypertext Markup Language, is the markup language used to create Web pages. HTML documents are saved in a .html format, and a Web browser interprets these documents in a way that the average user can understand.

Elements
HTML elements are what make up HTML documents. Each element is distinguished by using different HTML tags. There are two types of elements: inline and block. Inline elements are a single tag, and have a simple function, such as a line break. Block elements on the other hand require a closing tag at the end of the element, telling the Web browser when the element ends.
